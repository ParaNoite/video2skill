# Video2Skill 约束清单

## 产品约束

- 首版只做 Windows
- 首版只接受本地文件和 B 站链接
- 首版使用 CLI + 本地网页
- 首版主输出为 Markdown

## 合规约束

- 不绕过付费、会员、地区、验证码或访问控制
- 不保存 Cookie、令牌、授权凭据
- 平台适配器必须可关闭、可回退

## 内容约束

- 命令、路径、版本号等高风险信息必须有证据
- 没有证据的内容只能进入待确认区
- `KnowledgeUnit` 不允许凭空生成事实

## 工程约束

- 只通过契约层跨模块协作
- 每个 feature 必须可单独测试
- 每个 feature 必须可单独回退
- 任务目录、缓存目录、输出目录必须分离

## 处理约束

- 无字幕时允许 ASR 回退
- OCR 首版先保留接口和数据结构
- 商业模型 API 不在首版范围
- 桌面壳不在首版范围

## 失败约束

- 输入不支持时必须明确失败原因
- 不可用来源必须降级或关闭，不得静默继续
- 低置信度内容必须显式标记
