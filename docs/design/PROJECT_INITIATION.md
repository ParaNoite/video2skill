# Project Initiation

Source: `Video2Skill_Local_Agent_总体技术报告与可行性分析_V1.0.docx`

## Objective

Build a local-first agent that turns instructional videos into structured
notes, technical documents, and optional Skill packages.

## Initial scope

- local file ingestion
- platform adapter layer
- evidence fusion and knowledge extraction
- markdown / docx / Skill rendering
- audit and task manifest tracking

## Phase A

Freeze requirements, create the code skeleton, and make local-file task
creation work end to end.
